How to use:

1. Download .rar file.
2. Extract file.
3. Copy and paste the entire folder in local Server folder.(C:\wamp\www) if using wamp server.
4. Start server.
5. Open Browser, type in address bar localhost/foldername.
6. Now you can use chat application.
