How to use:

1. Download .rar file.
2. Extract file.
3. Import database in local server.
4. Copy and paste the entire folder in local Server folder.(C:\wamp\www) if using wamp server.
5. Start server.
6. Open Browser, type in address bar localhost/foldername.
7. Now you can use chat application.
